# Rick And Morty

Небольшой проект для взаимодействия с API RickAndMorty - [https://rickandmortyapi.com/], где реализованы возможности взаимодействия с GET запросами для получения списка персонажей одноименного мультика, а так же информации и каждом доступном персонаже по отдельности.

## В проекте реализовано:

1. Взаимодействие с API через GET методы;
2. Получение списка персонажей;
3. Просмотр подробной информации о персонаже;
4. Редактирование информации о персонаже (локально, т.к API не предоставляет возможность сохранения на сервер);
5. Infinite scroll;
6. Фильтрация;

## Стэк:

React, Typescript, React Router, Tailwind.

## Проект доступен по ссылке - [https://alexyuryevit.github.io/rick-and-morty/]

## Локальный запуск проекта:

git clone https://github.com/alexyuryevit/rick-and-morty.git
cd rick-and-morty

### Установить зависимости

yarn install

### Запустить в режиме разработки (localhost:5173)

yarn dev
